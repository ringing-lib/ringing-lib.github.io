Contributing to the Ringing Class Library

If you think the Ringing Class Library is a good idea and would like to contribute to its development, you are very welcome to help. You will need to get an ID on SourceForge and then visit the project page, which contains links to all sorts of information related to the project. If you email a project administrator, they'll add you to the list of developers and you will then be able to get at and change the source code.

If you don't want to get involved with developing the library, but would like to report a bug or make a suggestion for future developments, please visit the project page where you can access various forums and submit bugs.
