Contributing to the Ringing Class Library

If you think the Ringing Class Library is a good idea and would like to contribute to its development, you are very welcome to help. You will need to get an ID on GitHub and then visit the project page, which contains links to all sorts of information related to the project. If you'd like to contribute then please fork the repository and then issue a pull request.

If you don't want to get involved with developing the library, but would like to report a bug or make a suggestion for future developments, please visit the project page where you can access various forums and submit bugs.
