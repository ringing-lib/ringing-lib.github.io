Read the manual

This is the manual for the Ringing Class Library, version 0.3.0. It is not automatically kept up to date, but should always be fairly recent. The most recent version of the manual can be downloaded from SourceForge.

You can get the manual in several formats:

  • HTML (one file per node)
  • HTML (single file)
  • PDF
The manual is under development at the same time as the library, so parts are not totally finished.
